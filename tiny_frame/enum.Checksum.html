Checksum types.

Variants

None

The no-op checksum type.

Xor

This checksum type will return the bit-flipped xor of all the bytes in the buffer.

Crc16

This checksum type will return the CRC16 sum with the polynomial 0x8005 (x^16 + x^15 + x^2 + 1).

Crc32

This checksum type will return the CRC32 sum with the polynomial 0xedb88320.
