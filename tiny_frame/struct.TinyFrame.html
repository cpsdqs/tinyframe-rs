A TinyFrame instance.

Len is the length field type, ID is the ID field type, and Type is the type field type. You may use custom types, but in general you should pick from u8, u16, u32, and u64.

Fields

peer_bit: Peer

The peer bit.

parser_timeout: Option<Ticks>

The parser timeout after which the parser will be reset.

sof_byte: Option<u8>

The optional start-of-header byte.

chunk_size: usize

The chunk size. 1024 by default.

cksum: Checksum

The checksum type. Xor by default.

write: Option<Box<Fn(&mut TinyFrame<Len, ID, Type>, &[u8])>>

A function called every time something is written. This must be implemented.

claim_tx: Option<Box<Fn(&TinyFrame<Len, ID, Type>)>>

A function called before writing, for claiming the TX interface.

release_tx: Option<Box<Fn(&TinyFrame<Len, ID, Type>)>>

A function called after writing, for releasing the TX interface.

Adds an ID listener.

The listener will be called if a message with the specified ID is received. If timeout is not None, the listener will expire after the specified number of ticks.

Note that if the returned IDListener is dropped, the listener is too.
